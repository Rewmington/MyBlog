仓库地址：本项目的仓库地址（SSH）是git@github.com:Rewmington/MyBlog.git

背景：此项目用于制作我的个人博客，后面会优先给我自己使用，后面会开放给其他身边成员，这是一个纯静态无后端的网页项目

“请帮我用 React + Tailwind CSS + Framer Motion 编写一个高颜值的个人博客首页。这是一个纯前端项目，准备部署在 GitHub Pages 上。
1. 整体视觉风格：
二次元/宅系审美： 背景使用淡紫色和粉色的梦幻渐变，并实现动态的‘樱花飘落’全屏粒子效果。
毛玻璃拟态 (Glassmorphism)： 所有卡片必须是半透明的（bg-white/30），带有强烈的背景模糊（backdrop-blur-lg）和细微的白色高光边框（border-white/50）。
便当盒布局 (Bento Grid)： 页面主体由多个错落有致、大小不一的圆角矩形卡片组成，圆角跨度要大（rounded-3xl）。
2. 核心功能组件：
顶部： 一个极简的半透明圆角搜索框。
个人信息卡： 包含头像、昵称、一段描述文字、社交图标链接以及简单的文章/照片统计数字。
音乐播放器卡： 显示专辑封面（旋转动画）、歌名、进度条以及播放控制按钮。
Bento 文章展示： 至少展示 3-4 个不同尺寸的文章预览块，有的带大背景图，有的带文字蒙层。
底部状态栏： 黑色半透明长条，左侧显示‘15:35:17’这种实时更新的时钟，中间显示‘系统已运行 XX 天’，右侧显示技术栈图标（Next.js, React, Tailwind）。
3. 高级交互动效：
3D 倾斜效果： 鼠标移动到任何卡片上时，卡片要根据鼠标位置产生平滑的 3D 旋转倾斜感（使用 Framer Motion 实现）。
聚光灯效果： 鼠标在卡片上移动时，卡片内部应有一个跟随鼠标的微弱高光圆圈。
入场动画： 页面加载时，所有卡片按照从左到右、从下到上的顺序错落弹出。
4. 开发要求：
纯前端： 不要使用任何后端 API，所有数据（文章列表、运行时间）都写死在代码常量里。
响应式： 确保在手机端时，Bento 布局会自动坍缩为单列排列。
代码完整性： 请直接给出完整的单文件代码或主要组件代码，确保我复制后能直接运行看到效果。”
